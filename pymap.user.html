pymap.user¶

class pymap.user.UserMetadata(config, *, password=None, **params)¶

Contains user metadata such as the password or hash.

Parameters

  • config (IMAPConfig) – The configuration object.

  • password (Optional[str]) – The password string or hash digest.

  • params (Optional[str]) – Metadata parameters associated with the user.

property role: Optional[str]¶

Return type

Optional[str]
